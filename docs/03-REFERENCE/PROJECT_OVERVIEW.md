# Project Overview

This local `Docs/` folder contains project organization notes and run instructions.

- `ai-job-automation/` holds the main Python automation pipeline.
- `roles/role_profiles.yaml` defines role-specific scraping and scoring behavior.
- `scrapers/` contains source-specific and fallback scraping logic.
- `filters/` defines post-scrape filtering rules.
- `.gitignore` is configured to ignore `Docs/`, `*.md`, and `*.env` files so documentation and secrets are not pushed.
